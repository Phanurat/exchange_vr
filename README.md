# exchange_vr

## ข้อสอบ Backend
##### โจทย์ข้อ 1 ให้ออกแบบระบบฐานข้อมูล(ทำเป็นรูปแบบ ER) ที่เป็นตัวกลางของการแลกเปลี่ยน Cryptocurrencies เช่น Bitcoin โดย สามารถนำเงิน Fiat (THB,USD) มาซื้อเหรียญ จาก User คนอื่นๆในระบบได้ และสามารถจะโอน เหรียญหากันภายในระบบ หรือ โอนหาคนอื่นภายนอกระบบได้ ยกตัวอย่าง https://c2c.binance.com/th/trade/buy/BTC
